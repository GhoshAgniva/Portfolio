Verilog Traffic Light System for T-Intersection on FPGA

The Design and Implementation of Traffic Light Controller for T-Intersection on FPGA project involves creating a reliable and efficient traffic control system specifically for a T-shaped intersection, using Verilog HDL and FPGA technology.
